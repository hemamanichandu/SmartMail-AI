Smart Mail AI – Intelligent Email Assistant
Overview

Smart Mail AI is an AI-powered backend application that generates professional email replies, subject lines, and job application emails automatically using Spring AI.

Features
Generate professional email replies based on input
Create short and relevant subject lines
Send automated job application emails with resume
Ensures polite, structured, and clear communication

Use Case
Helps users respond to emails quickly with proper tone and clarity, reducing manual effort and improving communication efficiency.

Tech Stack
Java
Spring Boot
Spring AI (Ollama)
JavaMailSender
REST APIs

Sample Output :

Input:
“Can you send me your resume for Java Developer role?”

Generated Reply:
“Thank you for reaching out. I am interested in the opportunity and have attached my resume for your review…”
